# Statistical Thinking for Data Science

This project focuses on **statistical reasoning and interpretation**, not just formulas or model scores.

The goal is to demonstrate how core statistics concepts are applied in real Data Science workflows when interpreting metrics, uncertainty, and experiments.

Rather than optimizing models, this repository emphasizes **how statistical thinking guides decisions**.

---

## 📌 Why This Project

In practice:
- Metrics are estimates, not absolute truths
- Results vary due to sampling
- Uncertainty matters as much as averages
- Decisions depend on confidence, not just numbers

This project explores these ideas through simulations and visualizations using Python.

---

## 📊 Topics Covered

Each notebook focuses on one core statistical concept and its real-world implication:

1. **Central Limit Theorem**
   - Why estimates behave normally
   - Why confidence intervals work

2. **Sampling Variability**
   - Why metrics change across runs
   - Why one evaluation is not enough

3. **Confidence Intervals**
   - Measuring uncertainty around metrics
   - Comparing models responsibly

4. **Statistical vs Practical Significance**
   - When “significant” results don’t matter

5. **A/B Testing Intuition**
   - Confidence, variability, and decision-making

6. **Bias vs Variance**
   - Understanding generalization behavior

7. **Correlation vs Causation**
   - Avoiding misleading conclusions
